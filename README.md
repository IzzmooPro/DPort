# DPort

DPort is a small Windows tool that helps Discord open reliably by preparing the required DNS and local connection path from one interface.

## Download

Use the latest installer from GitHub Releases:

`DPort-Setup-<version>.exe`

## Updates

DPort checks GitHub Releases on startup. If a newer release is available and it includes a setup `.exe` asset, DPort can download and start the installer for you.
